Teamcli

Like Ctrl+R on steroids for your terminal

🚀 Features

  • 🔍 Fuzzy search your frequently used commands
  • 📝 Save commands with optional description
  • ❯_ Run parameterized commands using templates — great for secrets & variables (coming soon)
  • 🌐 Sync your commands across machines — same commands, anywhere you code (premium)
  • 👥 Share commands with your team — no more digging through Notion docs or Slack threads (premium)

📦 Installation

npm install -g tcli

📘 Usage

  • teamcli --add → Add a new command
  • teamcli → Search and run a saved command
  • teamcli --import ./file.json → Import commands
  • teamcli --export ./file.json → Export commands
